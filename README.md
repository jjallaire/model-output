# Model Output

Install for a project by executing this from the project working directory:

```bash
$ quarto add jjallaire/model-output
```
Example usage:

````markdown

```{.yaml .model-output}
- system: >
    You are a helpful assistant.
- user: >
    What is your name?
- assistant: >
    My name is Claude.
- separator
- assistant: >
    My name is Claude.
```
````
